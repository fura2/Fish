## Usage

- ``W`` or ``Up`` key : go up
- ``S`` or ``Down`` key : go down
- ``A`` or ``Left`` key : go left
- ``D`` or ``Right`` key : go right
- ``Q`` key : go backward
- ``E`` key : go forward
